# Quickstart: Wedding Dashboard

**Feature Branch**: `004-wedding-dashboard`
**Created**: 2026-01-13

## Prerequisites

1. VowSync development environment set up
2. Supabase project running with seed data
3. Node.js 18+ installed

## Quick Verification Steps

### 1. Start Development Server

```bash
cd vowsync
npm run dev
```

### 2. Navigate to Dashboard

1. Open browser to `http://localhost:5173`
2. View the wedding list
3. Click on a wedding card
4. You should see the Wedding Dashboard

### 3. Verify Dashboard Sections

| Section | Expected Behavior |
|---------|------------------|
| **Metrics Grid** | 4 cards showing: Days until wedding, Guest count, Events, Budget |
| **Events Summary** | List of up to 5 events in chronological order |
| **Quick Actions** | 4 buttons: Add Guest, Add Event, View Timeline, Manage Budget |
| **RSVP Progress** | Progress bar with confirmed/declined/pending breakdown |
| **Activity Feed** | List of up to 5 recent activities |

### 4. Test Navigation

| Action | Expected Result |
|--------|-----------------|
| Click "View Timeline" | Navigate to `/weddings/:id/events` |
| Click "Add Event" | Navigate to `/weddings/:id/events/new` |
| Click event in summary | Navigate to `/weddings/:id/events/:eventId/edit` |
| Click back button | Return to wedding list |

### 5. Test Edge Cases

| Scenario | How to Test | Expected |
|----------|-------------|----------|
| No events | Delete all events from wedding | "No events yet" empty state |
| Past wedding | Set wedding_date to past date | "X days ago" instead of countdown |
| Wedding today | Set wedding_date to today | "Wedding Day!" message |
| No budget set | Set budget_total to 0 | "Not set" instead of percentage |

## Common Issues

### Dashboard shows "Loading..." indefinitely

**Cause**: Missing RLS policy or auth token
**Fix**: Ensure user is authenticated and RLS policies allow access

### Guest counts show 0

**Cause**: No guests added to wedding
**Fix**: Add guests via `/weddings/:id/guests/new` (future feature) or directly in Supabase

### Activity feed empty

**Cause**: No activity_log entries for wedding
**Fix**: Activity logs are generated by triggers; make changes to wedding data

### Budget shows "Not set"

**Cause**: budget_total is 0 or null
**Fix**: Update wedding to set budget_total > 0
